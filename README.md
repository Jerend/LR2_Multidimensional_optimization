# LR2_Multidimensional_optimization

Установка зависимостей
```bash
pip install -r requirements.txt
```

Запуск
```bash
python main.py
```

Использование
```bash
1. Выбираем функцию

2. Задаем координаты начальной точки X1 по оси x1 и x2 

3. Задаем направление d1 по оси x1 и x2 

4. Задаем направление d2 по оси x1 и x2 

5. Задаем точность e

6. Задаем интервал для поиска лямбды

7. Запускаем алгоритм
```
